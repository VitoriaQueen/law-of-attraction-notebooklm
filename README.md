# 📚 AI Study with NotebookLM - Law of Attraction 

## 🎯 Context and Objectives

This project explores the use AI-assisted learning (NotebookLM) while approaching the Law of Attraction and related topics.

**Objectives:**
- AI Literacy
- Hands-on practice with the NotebookLM tool.
- Prompt Engineering
- Creation of reusable prompts for future studies.
- Curation of sources.
- Critical Thinking
- Information Synthesis
- Creation of a short study guide.
- Technical Writing (Markdown)
- Documentation Skills
- Conceptual Understanding (Law of Attraction)
- Presentation & Portfolio Building
- Self-Learning with AI
